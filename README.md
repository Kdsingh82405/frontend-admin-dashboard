# Frontend Admin Dashboard

A responsive admin dashboard built using React + TypeScript.

## Features

### Authentication
- Login validation
- Context API authentication
- Protected routes

### Dashboard
- Statistics cards
- API data fetching
- Loading handling

### Users Management
- View users table
- Add user
- Edit user
- Delete user
- Search users
- Sorting (A-Z / Z-A)
- Status filtering
- Pagination

### Performance Optimization
- Lazy loading (code splitting)
- useMemo optimization
- React.memo to prevent re-renders

### Responsive Design
- Mobile friendly layout
- Tablet & desktop optimized
- Scrollable tables

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- React Router
- Axios
- Context API

## Folder Structure
src/
 ├── api/
 ├── components/
 ├── context/
 ├── pages/
 ├── routes/
 └── utils/
